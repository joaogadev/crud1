# 📚 Crud Simples de manipulação de Usuários

Projeto com o objetivo de consolidar os conhecimentos e aprendizados no desenvolvimento com SpringBoot, criação de CRUD's e utilização do Postman

---

# 🚀 Funcionalidades

* Cadastro de novos usuários
* Atualização das informações de usuários cadastrados
* Listagem de usuários geral, por email e por id
* Remoção do usuários já cadastrados

---

# 🛠 Tecnologias Utilizadas

* **Java**
* **Spring Boot**
* **SQL**
* **JPA**
* **Programação Orientada a Objetos (POO)**
* **Git e GitHub**

# ⚙ Como Executar o Projeto

### 1️⃣ Clonar o repositório

```bash
git clone https://github.com/joaogadev/Lumenoteca.git
```

### 2️⃣ Entrar na pasta do projeto

```bash
cd crud1
```

### 3️⃣ Acessar o Banco com o JPA

Insira suas infromações do banco de dados em src/main/resources/application.properties 

```
spring.datasource.url=jdbc:mysql://localhost:3306/bd1
spring.datasource.username=root
spring.datasource.password=joao@2020
```


### 4️⃣ Abrir em uma IDE Java

Pode ser executado em qualquer IDE:

* IntelliJ IDEA
* Eclipse
* NetBeans
* VS Code

### 5️⃣ Executar o programa

Execute a classe principal do projeto.
